Paper Matrix

Item Primary contribution Best use in our next project Combination strength
Dr-CiK Benchmark for agents that must retrieve forecast-relevant context from noisy corpora. Main forecasting/deep-research testbed. Highest: gives the task and failure modes.
MoiraiAgent Agentic contextual time-series forecasting with expert selection and tool orchestration. SOTA target and architecture to beat or stress-test. Highest: direct competitive baseline.
Moirai Universal time-series forecasting transformer. Numerical foundation-model baseline. Medium: useful baseline, less aligned with trust/security by itself.
GIFT-Eval General benchmark for zero-shot time-series forecasting. External benchmark context for SOTA claims. Medium: useful for positioning.
Context is Key Forecasting benchmark where textual context is essential. Predecessor to Dr-CiK; useful for ablations and data design. High: clean context/forecast interface.
SafePred Predictive guardrail using world-model future-risk prediction. Convert "forecasting" from target output into a safety mechanism. Highest: natural bridge between forecasting and agent safety.
SafeHarness Lifecycle-integrated defense layers across agent operation. Defense architecture for deep-research forecasting agents. High: maps well to retrieval, planning, tool use, and state update.
HarnessAudit Trajectory-level harness safety benchmark and audit framework. Add safety metrics beyond final forecast accuracy. Highest: low-hanging evaluation paper angle.
Agentic AI Survey Systematic attack/defense taxonomy. Threat-model and related-work backbone. Medium: framing rather than direct method.
Prompt Injection/CI Contextual-integrity argument that data-instruction separation is insufficient. Threat model for malicious forecasting context and evidence flow. High: directly targets context-heavy forecasting agents.
Jailbreak Scaling FLOPs-budget scaling curves for jailbreak attacks. Attack/defense evaluation methodology. High: can become "attack budget vs forecast degradation/safety violation."
Zero Trust for AI Agents Enterprise deployment framework and demand signal. Product/security motivation: permissions, identity, sandboxing, memory, compliance. High: helps prove demand is real.

Initial Ranking For Your Situation

  1. Best team-aligned low-hanging fruit: "HarnessAudit for forecasting/deep-research agents." It is evaluation-heavy, publishable, and aligned with agent trust/security.
  2. Best algorithmic bridge: "SafePred-style predictive guardrails for forecast-driven agents." It reframes forecasting as a safety primitive.
  3. Best SOTA-beating path: "Robust MoiraiAgent-style expert/context selection under untrusted context." It still targets forecasting quality but adds a security dimension.
  4. Best demand story: "Zero Trust contextual forecasting agents for enterprise decisions." It ties to regulated data, tool permissions, provenance, and operational risk.
