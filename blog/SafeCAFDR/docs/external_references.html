External References

This page records the source families used to construct the SafeCAFDR research direction: forecasting agents, predictive guardrails, agent harness security, prompt injection, jailbreak scaling, zero-trust deployment, and agentic AI security surveys.

Forecasting and contextual agents

  • Dr-CiK: context-informed forecasting manuscript and immediate local motivation.
  • Moirai / MoiraiAgent: time-series foundation model and agentic forecasting baseline family.
  • GIFT-Eval: benchmark lineage for generalized forecasting evaluation.
  • Context is Key: evidence for contextual information improving time-series forecasting.

Predictive guardrails and future risk

  • SafePred: motivates predicting future unsafe trajectories rather than only filtering present outputs.

Harness security and audit

  • SafeHarness: practical security framing for tool-using agents and execution harnesses.
  • HarnessAudit: audit-oriented view of agent harness and tool risks.

Prompt injection and jailbreak scaling

  • Contextual-integrity prompt injection: threat model for retrieved content and long-context agents.
  • Jailbreak scaling: motivates scaling-law style evaluation of adversarial robustness.

Enterprise zero trust and deployment

  • Claude Zero Trust for AI Agents: deployment framing around least privilege, isolation, policy enforcement, and continuous verification.

Surveys and landscape

  • Agentic AI attack/defense survey: broad taxonomy for positioning SafeCAFDR among agent security work.
