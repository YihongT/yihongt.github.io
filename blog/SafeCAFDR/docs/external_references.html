External References (newly added, 2026-06-11)

Real, citable works gathered to harden the motivation in refined_proposal.md. These complement the internal library (Dr-CiK, MoiraiAgent, SafePred, SafeHarness, HarnessAudit, etc.) indexed in manifest.json. Grouped by the motivation link they support.

Context-dependence is the SOTA path (§2.1)

  • [CiK] Context is Key: A Benchmark for Forecasting with Essential Textual Information. arXiv:2410.18959 (ICML 2025). 71 tasks where text is required to forecast at all. https://arxiv.org/abs/2410.18959
  • [LLM-TSF-Rethink] Rethinking the Role of LLMs in Time Series Forecasting. arXiv:2602.14744. LLMs help mainly when they add complementary textual knowledge; otherwise simple attention matches them. https://arxiv.org/abs/2602.14744
  • [Modality-Gap] Overcoming the Modality Gap in Context-Aided Forecasting. arXiv:2603.12451. Aligning text+series modalities is the reliability bottleneck. https://arxiv.org/abs/2603.12451

The context channel is a proven attack surface (§2.2)

  • [PoisonedRAG] PoisonedRAG: Knowledge Corruption Attacks to RAG. arXiv:2402.07867 (USENIX Security 2025). 5 injected docs → ~90% attack success in a million-doc corpus; defenses insufficient. https://arxiv.org/abs/2402.07867
  • [AgentDojo] AgentDojo: A Dynamic Environment to Evaluate Prompt Injection Attacks and Defenses for LLM Agents. (NeurIPS 2024). Tool-using agents hijacked via malicious tool outputs; defines utility-vs-attack-success metrics. https://github.com/ethz-spylab/agentdojo
  • [IPI-Competition] How Vulnerable Are AI Agents to Indirect Prompt Injections? Insights from a Large-Scale Public Competition. arXiv:2603.15714. Real-world red-teaming confirms broad agent vulnerability. https://arxiv.org/abs/2603.15714
  • [PromptInj-CI] AI Agents May Always Fall for Prompt Injections (internal library, arXiv:2605.17634). Contextual-integrity impossibility result for data/instruction separation.

Honest context is often temporally invalid (§2.3)

  • [Lookahead-LLM] A Test of Lookahead Bias in LLM Forecasts. arXiv:2512.23847. "Lookahead Propensity" measure; apparent skill partly reflects pretraining leakage. https://arxiv.org/abs/2512.23847
  • [HoH] HoH: A Dynamic Benchmark for Evaluating the Impact of Outdated Information on RAG. arXiv:2503.04800. https://arxiv.org/abs/2503.04800
  • [STALE] STALE: Can LLM Agents Know When Their Memories Are No Longer Valid? arXiv:2605.06527. https://arxiv.org/abs/2605.06527

Provenance is often fabricated, and flows leak (§2.4)

  • [Cite-Faithfulness] Correctness is not Faithfulness in Retrieval Augmented Generation Attributions. arXiv:2412.18004 (SIGIR ICTIR 2025). Up to 57% of citations are post-rationalized. https://arxiv.org/abs/2412.18004
  • [AgentLeak] AgentLeak: A Full-Stack Benchmark for Privacy Leakage in Multi-Agent LLM Systems. arXiv:2602.11510. CI-grounded info-flow leakage benchmark. https://arxiv.org/abs/2602.11510
  • [MAGPIE] MAGPIE: A Benchmark for Multi-Agent Contextual Privacy Evaluation. arXiv:2510.15186. SOTA agents leak 35–51% of sensitive fields. https://arxiv.org/abs/2510.15186

The stakes are quantified (§2.5)

  • [Fin-Misinfo] Financial Misinformation and Trading Manipulation Using LLMs. Springer 2025. Survey of market-moving misinformation (e.g., ~$130B from a single false report). https://link.springer.com/chapter/10.1007/978-981-95-0433-6_1
  • [Adversarial-Headlines] Adversarial News and Lost Profits: Manipulating Headlines in LLM-Driven Algorithmic Trading. arXiv:2601.13082. Up to 17.7pp annual return loss from imperceptible headline edits. https://arxiv.org/abs/2601.13082
  • [NPR-Misinfo] NPR (2025-10-17), Financial markets are being subjected to misinformation — spread by AI. https://www.npr.org/2025/10/17/nx-s1-5575695/

Methodology / defense references

Agent trust & security — post-Feb 2026

  • [Trustworthy-Agentic-Survey] Towards Trustworthy Agentic AI: A Comprehensive Survey of Safety, Robustness, Privacy, and System Security. arXiv:2605.23989. Maps risk across the full agent workflow; flags temporal staleness + runtime uncertainty as evaluation pitfalls. https://arxiv.org/abs/2605.23989
  • [Layered-Attack-Survey] A Systematic Survey of Security Threats and Defenses in LLM-Based AI Agents: A Layered Attack Surface Framework. arXiv:2604.23338. https://arxiv.org/abs/2604.23338
  • [Formalizing-Agent-Security] A Framework for Formalizing LLM Agent Security. arXiv:2603.19469. Four contextual-security properties — Source Authorization, Task Alignment, Action Alignment, Data Isolation — the formal backbone our metric suite maps onto. https://arxiv.org/abs/2603.19469
  • [CausalArmor] CausalArmor: Efficient Indirect Prompt Injection Guardrails via Causal Attribution. arXiv:2602.07918. https://arxiv.org/abs/2602.07918
  • [LlamaFirewall] LlamaFirewall: An Open-Source Guardrail System for Building Secure AI Agents. arXiv:2505.03574 (Meta, production). PromptGuard 2 + Agent Alignment Checks + CodeShield. https://arxiv.org/abs/2505.03574
  • [MCP-Threat-Model] Model Context Protocol Threat Modeling and Analyzing Vulnerabilities to Prompt Injection with Tool Poisoning. arXiv:2603.22489. Poisoned tool descriptions/metadata the model reads but the user never sees; cross-tool harvesting up to ~75%. https://arxiv.org/abs/2603.22489
  • [Agentland-Backdoor] Malice in Agentland: Down the Rabbit Hole of Backdoors in the AI Supply Chain. Boisvert et al. (ServiceNow Research / MILA), arXiv:2510.05159 (CAIS 2026). Training-time / supply-chain backdoors (distinct from inference-time corpus poisoning): three threat models — direct finetuning-data poisoning, pre-backdoored base models, and novel environment poisoning of agentic training pipelines. Poisoning ~2% of collected traces embeds a trigger that makes the agent leak confidential info with >80% success; guardrail and weight-based defenses fail to detect it. https://arxiv.org/abs/2510.05159
