Paper Library

Created: 2026-06-11

This folder collects the forecasting and agent trust/security references for the "deep research for forecasting" next-step discussion.

Layout

  • pdfs/: canonical downloaded PDFs.
  • sources/: saved HTML sources for references that are blogs/project pages rather than papers.
  • by_theme/: categorized symlinks into pdfs/ and sources/.
  • manifest.json: machine-readable inventory with URLs, local paths, themes, and use cases.
  • classification.md: human-readable paper classification matrix.
  • research_opportunity_map.md: first-pass research combinations and demand checks.

Theme Index

  • 02_predictive_guardrails_and_future_risk: SafePred.
  • 03_harness_security_and_audit: SafeHarness, HarnessAudit, HarnessAudit project page.
  • 04_prompt_injection_and_jailbreak_scaling: Contextual-integrity prompt injection and jailbreak scaling.
  • 05_enterprise_zero_trust_and_deployment: Claude Zero Trust for AI Agents blog and ebook.
